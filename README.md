# frob-orientation-lean

A hybrid formalization of the Frobenius Orientation (Dream Theorem) for cubic periods of cyclotomic fields. 

This project establishes the orientation of the Frobenius automorphism for a randomized cubic primality test. It uses a **strong hybrid proof architecture** combining kernel-checked deductions in Lean 4 with exact algebraic witness generation via PARI/GP.

## Proof Architecture

The proof is split into two rigorous components:

1. **The Lean 4 Formalization (`proof2_frobenius_orientation_aristotle.lean`)**
   Establishes the universal deduction in $N$ given a valid $q$-witness. It proves the finite-field Frobenius conclusions by first establishing the Galois action on roots in the residue field of a cyclotomic extension, and then transporting that information back to the `AdjoinRoot(f)` model. The theorem is conditional on a sharply defined cyclotomic-period witness package (`h_valid_q`).

2. **The PARI/GP Certificate (`PARI_GP_Certificate_Proof.txt`)**
   Supplies the required witness package for specific primes $q$ by computing in the exact cyclotomic quotient ring (`Mod(x, polcyclo(q))`). It enforces the Gauss normalization $a \equiv 1 \pmod 3$, aligns the primitive root $g$ with $s_2$, and computes exact algebraic equalities (not floating-point approximations) for the period differences and polynomial identities required by the Lean theorem.

## Verified Primes

The original single-prime formalization verified $q = 2659$. Under the newly generalized hybrid architecture, the PARI/GP certificate generates and exactly verifies the cyclotomic witness for **all admissible primes up to $q = 97,039$**.
